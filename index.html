How to use this

Open the day, follow the timeline, adjust only when traffic says so.

The plan avoids midday hotel returns unless truly needed. Breaks are built into lunch, coffee, scenic pauses, and lower-effort stops so the day flows as one sensible route.

Trip Rules

  • Check Google Maps every morning before leaving.
  • Keep each walk to 30-60 minutes with seating nearby.
  • Eat near the final route, then return once for real rest.
  • Skip optional stops when parking looks annoying.
